## 📚 Estrutura

| Branch   | Descrição        | Tags                          |
|----------|------------------|-------------------------------|
| modulo01 | Módulo 01        | `aula01-mod01` ... `aula0N-mod01` |
| modulo02 | Módulo 02        | `aula01-mod02` ... `aula0N-mod02` |
| modulo03 | Módulo 03        | `aula01-mod03` ... `aula0N-mod03` |
| main     | Versão estável   | latest merge from modules     |

## 🚀 Como navegar

### Ver todas as tags (aulas)
git tag

### Ver código de uma aula específica
git checkout aula02-mod01

### Ir para um módulo inteiro
git checkout modulo01
